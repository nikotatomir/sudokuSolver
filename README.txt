An N x N sudoku puzzle with sqrt(N) x sqrt(N) subgrids is solved using the backtracking
algorithm.

The Python code was tested for 4x4, 9x9 and 16x16 grids and the results are located in the 
results/ folder (the corresponding puzzles are located in the sudokuPuzzles/ folder). 
	
Package requirements for running the simulations in a Conda virtual environment --> requirements.yml file.
